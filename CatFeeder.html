Cat Feeder Project

For my Intro to Microcontrollers for the Real World course, I was tasked to design the control circuit for a cat feeder that was provided by the professor. Along with designing the ciruit and choosing components, I had to design a PCB to house the electronics, and program the microcontroller.

Power

Plugs into the wall with a 12V DC adapter. It also saves time when not powered in the case of an outage.

Microcontroller

Using an Arduino Nano as a brain, and programmed using Arduino IDE, the catfeeder is a simple yet smart product that can perform multiple tasks.

Graphics

Users can interact with the product through the use of buttons and a graphical interface. The screen allows the user to see the time and mode they are on.

Reliability

The cat feeder rotates at a time set by the user without having to be refilled for 3 days. Perfect for short trips out of town.

Ease of Use

The instructions are straightforward- plug in the product and watch it go! The user can set the time and the daily alarm using the menu button and arrow keys.

Hard to Miss

With a combination of the lit-up I2C screen and size of the feeder, it is hard to miss on the floor.

Mechanical

The cat feeder's mechanical components consist of a stepper motor, a gear tray that colds 3 feeding cups, a lid with an opening to expose a single tray at a time, and a shell to cover the electrical components. 

This portion was provided by the professor, who took the role as a customer. Provided the mechanical shell of the cat feeder, a control circuit and PCB had to be designed to fit within the the cat feeder. There were already 4 built-in mounting holes inside of the feeder in addition to 3 buttons and a opening for the display. This mean't the PCB had to be designed to mount onto those holes and allow for the necessary componenets to be exposed through the top lid of the feeder.

Electrical

The cat feeder is powered by a 12 Volt DC wall plug. It contains a PCB that houses an Arduino, motor driver, and additional connections for power, the stepper motor, and the homing sensor. It also contains a I2C display and 3 push buttons. Using a buck converter, the 12 V input power is converted down to 5V, allowing for the PCB to power the Arduino,homing sensor, and clock.

Schematic of the control circuit for the cat feeder

After the control circuit was designed, a PCB was designed. The main constraint of the PCB was for it to mount onto the 4 pre-defined mounting holes, and for the buttons and I2C display to go through the pre-defined openings of the caddy shield. The DC input port should also align with the opening on the caddy such that a jack can connect to the cat feeder while the cat feeder is assembled. The final design of the PDB ended up being 2 layers, with a ground plane on the botton, and two power planes on the top- one for 5V and one for 12 V. 

Front of PCB

The top of the PCB. The top of the PCB has the I2C display and the buttons mounted. The other components are soldered to the bottom of the PCB due to a height constraint caused by the caddy cover.

Back of PCB

The bottom of the PCB. A bulk of the components are attached onto the bottom of the PCB due to the height constraint caused by the cover of the caddy. The bottom features the power port, capacitors resistors, molex connectors, the motor driver, Arduino microcontroller, and the real time clock. The ground plane is also on the back of the board

 Software

The microcontroller was programmed using Arduino IDE. Arduino is the brains of the cat feeder. It supports the following features:

  • Change/Set time: The user can set the time of the internal clock
  • Set Alarm: The user can set the daily alarm for the cat feeder to rotate
  • Rotate Motor: Controls the stepper motor rotation and speed. Also determines if motor is in home position.
  • Display: Displays the necessary information onto the I2C device
  • Button Inputs: Takes mechanical button inputs into the system.

The user interacts with the cat feeder using the buttons that are located on the front panel next to the display. The user can use these buttons to change the time, set the alarm time, and toggle between the different modes of the cat feeder. As long as the cat feeder is powered by the 12 Volt power supply, the user can interact with the microcontroller. When the device is unplugged it will keep track of the time, though the user can still choose to re-adjust the clock if need be.

In order to ensure that the cat feeder is positioned such that a single food tray is exposed at a time, a OPB740WZ homing sensor was used. When the enableMotor() function is called, the motor rotates the food tray until the homing sensor reads that it is positioned across the white paint. The motor does not need to stop at any other position except when over the paint, which is positioned such that one of the three trays are exposed through the cat feeder cover. With that, only one motor control function was necessary.
